Introduction

This documentation aims to assist you with familiarizing yourself with the Electric Games Api. Here you can see what quieries that are avaialable and how to use them.

BaseURLS

Base URLS accommodate the information from all available resources within the API

GET https://localhost:7226/game

GET https://localhost:7226/consoles

GET https://localhost:7226/gameCharacter

Games schema

"Games search by id" https://localhost:7226/Game/{type the {id} here}

"Games search by name" https://localhost:7226/Game/GetByName?nameFromDb={type game {name} here}

"Game posting" "tips you can post from swagger" https://localhost:7226/game

"Games Update" "tips you can update from swagger" https://localhost:7226/Game

"Games Delete" "tips you can delete it from swagger" https://localhost:7226/Game/

Console schema

"Console search by id" https://localhost:7226/consoles/{type the {id} for console here}

"Console search by name" https://localhost:7226/consoles/GetByName?nameFromDb={type console {name} here}

"Console posting" "tips you can post from swagger" https://localhost:7226/consoles

"Console Update" "tips you can update from swagger" https://localhost:7226/consoles

" Console Delete" "tips you can delete it from swagger" https://localhost:7226/consoles/
